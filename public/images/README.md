# Images

Add your own images here for the landing page. The app currently uses placeholder URLs from picsum.photos so it runs without local files.

To use your own images:

1. Add files such as:
   - `hero-bg.jpg` – Hero background (farm field)
   - `about-farmers.jpg` – About section
   - `why-us-crops.jpg` – Why Us section
   - `testimonials-leaf.jpg` – Testimonials background
   - `contact-bag.jpg` – Contact section

2. In `lib/images.ts`, replace the `IMAGES` object to use local paths, e.g.:
   - `hero: '/images/hero-bg.jpg'`
   - etc.
